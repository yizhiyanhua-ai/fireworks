# 一支烟花 AI 社区官网

这是“一支烟花 AI 社区”的 GitHub Pages 官网项目，目标发布地址：

```text
https://yizhiyanhua-ai.github.io/fireworks/
```

## 技术栈

- Vite + React
- `base: '/fireworks/'`
- GitHub Pages Actions workflow

## 本地运行

```bash
npm install
npm run dev
npm run build
```

## 隐私边界

官网不展示私密群聊、个人联系方式、未授权成员信息、凭据、内部运营材料或登录后才能读取的正文。

## 发布

推送到 `yizhiyanhua-ai/fireworks` 的 `main` 分支后，GitHub Actions 会构建 `dist` 并部署 GitHub Pages。
